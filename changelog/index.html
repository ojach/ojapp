更新履歴（Changelog）

OJapp のアップデート内容をまとめています。

🔷 v1.0 — 2025/12/10

  • OJapp が正式リリース。アプリ生成がツール内で完全完結するように統合
  • 画像アップロード〜R2保存〜自動リサイズ〜manifest生成まで全工程を自動化
  • D1 データベースにより、アプリID・URL・アイコン情報を自動管理
  • 生成されるURLを ojach.com/ojapp/XXXXX 形式で統一
  • URLスキームに正式対応（Amazonアプリ等の起動が可能に）
  • 30秒カウントダウン画面を実装し、ホーム追加の導線を最適化
  • ゲストユーザーでも利用できる一般向けバージョンとして公開
  • UIを全面刷新し、アイコン選択→タイトル入力→URL入力→生成 の4ステップに最適化
  • 「使い方ガイド」「FAQ」「安全性ガイドライン」を公開し、サポート体制を整備
  • 内部構造を template 化し、今後のアップデート作業を効率化

🔷 v0.4 — 2025/12/05

  • 全ページで共通ヘッダー・フッターを導入
  • ダークモードに正式対応
  • 安全性ガイドラインページを追加
  • 使い方ガイドページを追加
  • ページ構造を head-template 化して作業効率が大幅向上

🔷 v0.3 — 2025/12/04

  • UI を全体的にアップデート（ボタン配置・カード幅調整）
  • ホーム画面追加を前提とした 30 秒カウントダウン機能追加
  • アイコンプレビュー機能を改善

🔷 v0.2 — 2025/12/03

  • アプリ生成機能の初期実装
  • manifest.json の自動生成機能を追加

🔷 v0.1 — 2025/12/02

  • OJapp 開発開始
  • 最初のビルダー UI を実装
  • GitHub Pages にて一般公開開始
