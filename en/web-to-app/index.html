How to Add a Website to Your Home Screen (iPhone / Android)

This guide explains how to add any website you’re viewing in your mobile browser as an icon on your home screen — just like an app.

And with OJapp, you can go even further: turn any URL into a custom “app” with your own icon.

How to Add a Website on iPhone (Safari)

  • Open the website in Safari
  • Tap the Share button (□↑)
  • Select “Add to Home Screen”
  • Enter a name → tap “Add”

How to Add a Website on Android (Chrome)

  • Open the website in Chrome
  • Tap the menu icon (⋮) in the top-right
  • Select “Add to Home screen”
  • Tap “Add” to finish

Does It Work Like an App?

Yes — once added, the website opens almost like a native app. Some sites even support **PWA installation**, enabling better performance and limited offline support.
