OJappは、日本在住の個人開発者 おじゃち が運営・管理しているWebサービスです。 本サービスは、個人によって企画・開発・運営されており、 「シンプルで使いやすいこと」を大切にしながら継続的な改善を行っています。

OJappについて

OJappは、Webサイトをホーム画面アプリとして利用できるようにするツールや、 日常的に使えるシンプルなWebユーティリティを提供しています。 面倒な設定やインストールを必要とせず、誰でも直感的に使える設計を目指しています。

運営方針

OJappでは、利用者目線を最優先に考え、 不要なデータ収集や過度な機能追加は行いません。 実際の利用状況やフィードバックをもとに、 必要な改善を少しずつ積み重ねていく方針で運営しています。

About the Operator

OJapp is operated and maintained by おじゃち, an independent developer based in Japan. This service is created and managed by a single individual, focusing on simplicity, usability, and continuous improvement.

About OJapp

OJapp is a web-based tool designed to make everyday web usage more convenient. Its main goal is to help users turn websites into home screen apps and manage simple web utilities without complex setup or installation.

Operating Policy

OJapp is developed with a user-first mindset. No unnecessary data is collected, and features are added only when they provide clear value. The service is continuously updated based on real usage and feedback.
